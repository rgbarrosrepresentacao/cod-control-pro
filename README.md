# COD Control Pro 📊

Sistema de gestão profissional para operações **Cash on Delivery (COD)**.

## Funcionalidades

- Dashboard com KPIs em tempo real
- Controle de Anúncios (Facebook, Google, YouTube, TikTok)
- Controle de Vendas (integração Logzz)
- Relatórios avançados com exportação PDF/Excel
- Simulador de Escala
- Meta mensal com barra de progresso
- Banco de dados na nuvem (Supabase)
